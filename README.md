Project: Inventory Policy under Demand Volatility

This project analyzes demand volatility across SKUs using the Walmart M5 dataset and evaluates how volatility influences the performance of fixed and dynamic reorder point policies.

Current progress:
Milestone 1 – Research design and data scope finalized
Milestone 2 – Demand volatility measurement and SKU segmentation

Dataset: M5 Forecasting Accuracy (Kaggle)
